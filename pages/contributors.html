Contributors

MYCODELAB is built and maintained by MSc Physics students committed to open scientific computing. Each contributor brings expertise in specific domains of computational physics.

How to Contribute

MYCODELAB welcomes contributions from students and researchers. Follow these steps to add your implementation.

  1. 01
    Fork the repository on GitHub and clone your fork locally.
  2. 02
    Create a new branch: git checkout -b feature/my-method
  3. 03
    Place your code in the appropriate category folder under codes/. Include a module docstring with method description and references.
  4. 04
    Add a README.md inside your folder explaining the algorithm and expected output.
  5. 05
    Open a Pull Request with a clear title. Review within 48 hours.
