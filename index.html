04 / Documentation

Getting Started

Everything you need to run, contribute, and extend this repository.

▶

How to Run Codes

Clone the repository and install dependencies:

bash 
git clone https://github.com/physicslab/repo.git
cd repo
pip install -r requirements.txt

Navigate to a category folder and run any script:

bash 
cd numerical_integration
python simpson_rule.py
⬡

Dependencies

numpy ≥ 1.24 Array operations, linear algebra
scipy ≥ 1.11 Scientific algorithms, integration
matplotlib ≥ 3.7 Plotting and visualization
astropy ≥ 5.3 Astronomy / astrophysics tools
Julia ≥ 1.9 High-performance numerical codes
⊞

Contribution Guide

  1. 01
    Fork the repository on GitHub and clone your fork locally.
  2. 02
    Create a new branch named after your feature: git checkout -b feature/my-method
  3. 03
    Place your code in the appropriate category folder. Include a docstring with method description and references.
  4. 04
    Add a brief README.md inside your folder explaining the algorithm and expected output.
  5. 05
    Open a Pull Request with a clear title and description. One of us will review within 48 hours.
